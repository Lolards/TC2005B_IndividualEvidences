Elementos que se utilizan en desarrollo Web:

Rutas absolutas y relativas

Las rutas absolutas son los enlaces completos, los cuales incluyen el dominio completo Ej: https://www.wikipedia.org. Las rutas relativas son el enlace sin el dominio, solo el orden de direcctorios Ej: ../paginas/contacto.html.

Rutas en el proyecto.

En el proyecto, el uso de las rutas es muy útil para organizar el uso de recursos. Una ruta absoluta se usa por ejemplo cuando se enlaza a un recurso fuera de nuestro dominio. La ruta relativa se usa cuando se enlaza a un recurso dentro de nuestro dominio. Solamente es la dirección de donde está este recurso. Ej: paginas/about.html

Navegación entre páginas

Existen varias formas de comunicarse con otras páginas, ya sean internas del proyecto o externas. A continuación se muestran los distintos enfoques.

Ir a Google (misma pestaña) : La etiqueta <a> sin atributo target abre el enlace en la misma pestaña, reemplazando la página actual.

Ir a Google (nueva pestaña) : Agregar target="_blank" abre el enlace en una pestaña nueva. Se añade rel="noopener noreferrer" por seguridad.


Botón type="submit" dentro de un <form>: usa el comportamiento nativo del navegador para navegar al action.

<button> independiente con JS: usa window.location.href para navegar.

Usa window.open(url, '_blank') para nueva pestaña desde JS.
