Preguntas de repaso — Fundamentos Web

1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una inmensa red de computadoras alrededor de todo el mundo conectadas entre sí.

En cambio, la web (la World Wide Web) es una enorme colección de páginas que se asienta sobre esa red de computadoras.

2. ¿Cuáles son las partes de un URL?

  • Protocolo
  • Nombre de host
  • Ruta
  • Cadena de consulta

3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Estandarizar la comunicación, permitiendo solicitar, crear, actualizar o eliminar información de forma uniforme y segura.

4. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

Se utiliza el método POST. Esto se debe a que POST envía los datos en el cuerpo de la petición en lugar de la URL, lo que ayuda a proteger información sensible como usuarios y contraseñas.

5. ¿Qué método HTTP se utiliza cuando se accede a una página a través de un URL?

El método GET. Solicita una representación de un recurso específico del servidor sin modificar su estado, siendo la forma estándar de visualizar contenido en la web.

6. Un servidor devuelve código 200. ¿Qué significa? ¿Ocurrió algún error?

Significa que la solicitud fue exitosa. No ocurrió ningún error.

7. Error 404: ¿es responsabilidad del desarrollador corregirlo?

Sí, ya que es responsabilidad del equipo técnico asegurar que la estructura del sitio y sus rutas sean correctas.

8. Error 500: ¿es responsabilidad del desarrollador corregirlo?

Sí, ya que un error 500 es un fallo interno del servidor. Significa que el código o la configuración del sitio se rompió. El usuario no tiene la culpa ni puede arreglarlo; solo alguien con acceso al código o al servidor puede identificar la causa y corregirla.

9. ¿Qué significa que un atributo HTML5 esté deprecated?

Significa que, aunque puede seguir funcionando en la mayoría de los navegadores actuales por compatibilidad, no se recomienda su uso en nuevos proyectos y es probable que desaparezca en el futuro.

Algunos elementos de HTML4 desaprobados en HTML5:

  • acronym
  • applet
  • basefont
  • big
  • center
  • dir
  • font
  • frame
  • frameset
  • noframes
  • strike
  • tt

10. ¿Qué componentes de estructura y estilo tiene una tabla?

Se compone de filas (horizontales), columnas (verticales) y celdas (intersección de ambas) que organizan los datos.

11. ¿Cuáles son los principales controles de un formulario HTML5?

Los principales controles permiten recopilar datos mediante:

  • input con tipos: text, email, number, checkbox, radio, date, password, color, range
  • textarea — para texto largo
  • select — menú desplegable
  • Botones: submit, reset, button

HTML5 también añadió atributos de validación como required, min, max y pattern.

12. ¿Cuál es el ciclo de vida y de desarrollo de los sistemas de información?

Un sistema de información es un conjunto de personas, máquinas y métodos organizados para recopilar, procesar y transmitir datos.

Su ciclo de vida comprende las siguientes etapas:

  1. Planificación: Se organizan tareas previas para asegurar el éxito del proyecto.
  2. Análisis: Se identifican y comprenden los requerimientos del sistema.
  3. Diseño: Se define la estructura y las posibles soluciones del sistema.
  4. Implementación: Se desarrolla el sistema usando herramientas y lenguajes adecuados.
  5. Pruebas: Se detectan y corrigen errores del sistema.
  6. Instalación o despliegue: Se prepara el entorno donde funcionará.
  7. Uso y mantenimiento: Se corrigen fallos, se adapta el sistema y se agregan mejoras.
