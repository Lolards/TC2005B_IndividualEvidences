Reflexión

¿Cuál es la diferencia entre padding y margin?

Padding es el espacio dentro del borde del elemento (entre contenido y borde). Margin es el espacio exterior (separación entre elementos).

¿Por qué se recomienda usar box-sizing: border-box?

Porque incluye padding y borde en el ancho/alto total del elemento, lo que hace los cálculos de diseño más predecibles y evita desbordes no deseados.

¿Qué significa diseñar con enfoque mobile-first?

Significa escribir estilos base para móviles primero y luego ampliar con media queries para tablet/desktop. Esto mejora rendimiento y mantenibilidad para usuarios móviles.

¿En qué situaciones preferirías rem sobre px?

Rem se prefiere para tipografía y espaciado cuando se busca escalabilidad y accesibilidad (ajustes globales de tamaño). Px sirve para bordes, sombras, etc.
